# Requirements

- [ox_lib](https://github.com/communityox/ox_lib)
- [qbx_core](github.com/qbox-project/qbx_core)
- [ox_inventory](https://github.com/communityox/ox_inventory)
- [ox_target](https://github.com/communityox/ox_target)

# Features

- Custom UI built with React + Lucide Icons
- Shopping cart system
- Multiple payment methods (cash or bank card)
- Money balance display
- Licensed items
- Job restricted shops
- Grade restricted items
- Item metadata support
- Locked item indicators showing why items are unavailable
- Themed UI colors for each shop
